# 手織り工房萌木 ホームページ要件定義書

このドキュメントは、沖縄にある「手織り工房萌木」のホームページをどのような目的で、どのように作るかをまとめたものです。

## 1. ホームページの目的
- 織（お）り物の魅力を、詳しくない人にも「わかりやすく」伝える。
- 沖縄の文化や「手織り工房萌木」の温かさを知ってもらう。
- パソコンでもスマホでも、オシャレで使いやすいページにする。

## 2. ターゲット（見てほしい人）
- 織物に興味（きょうみ）を持ち始めたばかりの人。
- 沖縄の工芸（こうげい）や体験（体験）について詳しくない観光客（かんこうきゃく）の人。
- シンプルでオシャレなものが好きな人。

## 3. ページ構成（こうせい）
- **トップページ (Home)**: 工房の第一印象を決める。綺麗な写真と温かいメッセージ。
- **織物について (About Weaving)**: 織物がどうやって作られるか、沖縄の歴史などを簡単に説明する。
- **工房・体験案内 (Workshop)**: 工房の場所や、手織り体験の内容を紹介する。
- **ギャラリー (Gallery)**: 素敵な作品の写真を見せる場所。
- **お問い合わせ (Contact)**: 質問や予約のための連絡先。
