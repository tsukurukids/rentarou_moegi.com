# 手織り工房萌木 ホームページ設計書

ホームページのデザインや、プログラムの動（うご）きに関する設計のまとめです。

## 1. デザインコンセプト
- **「静（しず）かな沖縄の風（かぜ）」**
- 白をたくさん使い、ゆったりした空間（くうかん）を感じさせる。
- 文字は細めで、圧迫感（あっぱくかん）のない上品（じょうひん）な見た目。

## 2. カラーパレット（色のルール）
- **ベースカラー**: ホワイト (#FFFFFF) ・・・ 清潔（せいけつ）感と広がり。
- **メインカラー**: サンドベージュ (#F5F5F0) ・・・ 砂浜（すなはま）のような優しい白。
- **アクセントカラー**: 藍（あい）色 (#1A237E) ・・・ 沖縄の海や織物の糸をイメージ。

## 3. フォント（文字）
- **種類（しゅるい）**: ゴシック体（游ゴシックやInter）を中心に、スッキリとしたもの。
- **太さ**: あまり太くせず、細めのウェイト (300〜400) を使う。

## 4. レスポンシブ設計
- **PC**: 横幅をたっぷり使った、写真の映（は）えるレイアウト。
- **スマホ**: 縦一列に並（なら）べ、指でタッチしやすい大きなボタン。

## 5. 技術スタック
- **プログラム**: HTML5, CSS3, JavaScript (Vanilla JS)。
- 外部のライブラリは使わず、ブラウザだけで動くシンプルな構成にする。
